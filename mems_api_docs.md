# MEMS API 文档

***

## 概述

本文档描述了MEMS应用的RESTful API接口和数据类型定义，方便AI Agent调用。

### 基础信息

- **基础URL**: `http://ip:port/api/v1/`
- **认证方式**: Access-Token（在请求头中携带）
- **Content-Type**: `application/json`

### 认证说明

所有API请求（除登录接口外）需要在请求头中携带Access-Token：

```http
Access-Token: <your_token>
```

### 调用示例

使用Python requests库调用示例：

```python
import requests

base_url = 'http://ip:port/api/v1'
headers = {'Access-Token': 'your_token'}

# 获取所有用户
response = requests.get(f'{base_url}/auth/users', headers=headers)
print(response.json())
```
